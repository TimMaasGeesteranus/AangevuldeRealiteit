# Opzet onderzoek
##	Onderzoeksvraag
Dit onderzoek is ontstaan uit de volgende hoofdvraag:

 *“Welke app/web framework is het meest geschikt om onze applicatie mee te realiseren?”*

Vanuit de hoofdvraag zijn er een aantal deelvragen ontstaan die uiteindelijk worden gebruikt om de hoofdvraag te kunnen beantwoorden. Per gevonden alternatief worden deze deelvragen beantwoord en worden de resultaten gebruikt om tot een eindoordeel te komen.

## Deelvragen
Om erachter te komen welk app/web framework het meest geschikt is voor ons project, gaat er onderzoek gedaan worden naar verschillende frameworks. Deze frameworks zullen worden getest op verschillende vlakken.

### **Hoe eenvoudig is het framework te implementeren?**
*Onderzoeksmethode: lab en bieb*

Om ondersteuning te bieden tijdens de uiteindelijke implementatie, is het belangrijk dat er wordt onderzocht hoe eenvoudig het is om een framework te implementeren in een applicatie. Hiervoor zal er een Lab test worden uitgevoerd.

### **Welke platformen worden er ondersteund?**
*Onderzoeksmethode: lab*

Om ervoor te zorgen dat de applicatie door een zo groot mogelijk publiek gebruikt kan worden, zal er onderzocht worden welke platformen ieder framework ondersteund. Hoe meer platformen er ondersteund worden, hoe meer potentiële gebruikers de applicatie kunnen gebruiken.

### **Hoe is de performance van het framework?**
*Onderzoeksmethode: bieb*

De snelheid van het framework speelt ook een rol in het maken van een keuze. Het is echter erg moeilijk om te toetsen welk framework het beste presteert. Daarom zal er zo veel mogelijk informatie worden opgezocht over ervaringen van andere gebruikers. Wanneer een framework veel gebruikt wordt, betekent dit meestal wel dat een framework goed presteert.

### **Welke functionaliteit biedt het framework?**
*Onderzoeksmethode: Werkplaats en bieb*

Het is belangrijk dat het framework voldoende functionaliteit biedt om de applicatie te kunnen realiseren. Doormiddel van het ontwikkelen van een prototype implementatie, kan er getest worden welke functionaliteiten het framework biedt.

### **Support & community**
*Onderzoeksmethode: bieb*

Er moet voldoende documentatie online te vinden zijn over het framework. Ook moet de community bestaan uit genoeg andere gebruikers die werken met het framework zodat er eventueel ondersteuning gevraagd kan worden.
