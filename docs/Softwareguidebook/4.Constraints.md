# Software Guidebook

## Constraints
In dit hoofdstuk worden de constraints, oftewel de beperkingen waar het development team rekening mee moet houden beschreven.



- De Visual Domotica Builder wordt getest op Chrome (versie 74.0) en Mozilla Forefox (66.0.5). Alleen op deze browsers is er zekerheid dat de VDB correct functioneert.
- De Visual Domotica Builder wordt geschreven in Javascript (er wordt gebruik gemaakt van het React framework), HTML en SCSS. Andere talen zullen niet gebruikt worden.
- De Visual Domotica Builder zal in het Engels worden geschreven, andere talen zullen niet ondersteund worden.
- Er is geen budget beschikbaar, dus zullen er geen aankopen worden gedaan voor ondersteunende software, hardware en informatie.

Er wordt in onze applicatie ook gebruik gemaakt van enkele dependencies die niet standaard in een React-Redux applicatie zitten. Hieronder vallen:
- Lodash-throttle:  V.4.1.1.  
Lodash-throttle wordt gebruikt in onze applicatie om performance te verbeteren, wij gebruiken het in de store zodat de store niet continu geüpdatet wordt. Als dit niet gedaan zou worden zouden verouderde laptops performance problemen ondervinden, ook zouden twee tabbladen niet correct met elkaar kunnen communiceren.

- Ajv: V.6.10.0.  
Ajv wordt in onze applicatie gebruikt om de te importeren configuratie te valideren. Er is een schema gedefinieerd waarmee JSON bestanden gevalideerd worden. Dit schema is [hier](https://github.com/HANICA-DWA/feb2019-project-capybara/blob/master/code/capybara/src/assets/schemas/validationSchema.js) te vinden 

- Dropbox: V.4.0.17.  
Dropbox wordt gebruikt met authentiseren van de gebruiker met OAuth2 via de Dropbox API. Daarnaast wordt het gebruikt voor het importeren en exporteren van projecten van en naar Dropbox.

- react-dropbox-chooser: V.0.0.5.
De react-dropbox-chooser wordt gebruikt om de file selector van Dropbox te gebruiken. Met deze file selector worden bestaden geïmporteerd uit Dropbox.

- Googleapis: V.40.0.0.  
Googleapis wordt gebruikt met authentiseren van de gebruiker met OAuth2 via de Google API. Daarnaast wordt het gebruikt voor het importeren en exporteren van projecten van en naar Google Drive.

- react-google-picker: V.0.1.0.
De react-google-picker wordt gebruikt om de file selector van Google te gebruiken. Met deze file selector worden bestanden geïmporteerd en geëxporteerd.

- Docker: V.1.0.0.  
Docker wordt gebruikt om de Dropbox en GoogleDrive apis op te draaien.

- Docker-compose: V.0.17.5.  
Docker-compose wordt in combinatie gebruikt met Docker om sneller Dockercontainers te kunnen maken.
